# سرویس مدیریت کارها با پایتون و FastAPI

این پروژه یک سرویس ساده مدیریت کارها (Task Management Service) است که برای درس مهندسی نرم‌افزار پیشرفته طراحی شده است.

## ویژگی‌ها

- ایجاد کار جدید
- مشاهده لیست کارها
- مشاهده یک کار
- ویرایش کار
- حذف کار
- پیاده‌سازی با FastAPI و پایتون
- مستندات خودکار در مسیر `/docs`

## اجرای محلی

```bash
pip install -r requirements.txt
uvicorn main:app --reload
